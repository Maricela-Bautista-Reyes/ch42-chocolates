Estructura basica de un HTML

La estructura básica de una página web se compone por las etiquetas "", "" y "". La primera indica el comienzo de la página; la segunda es la cabecera y contiene información descriptiva del sitio; y la tercera es la que abarca todos los elementos visibles dentro de la página.

Una estructura básica se vería asi:

body { font-family: Arial, sans-serif; line-height: 1.6; max-width: 600px; margin: 20px auto; padding: 0 20px; } h1 { color: #333; text-align: center; } h2 { color: #555; } p { margin-bottom: 15px; }
